Instructions

Experimental Feature! This system is a bit finicky and still a Work in Progress. Patience required - results may vary. Not responsible for any table-flipping incidents.

How to Use Celebrations

  1. Click + in the Characters header to add player characters
  2. Select a character by clicking their portrait
  3. Click 🎉 to add a celebration type (Attack Roll, Damage Roll, etc.)
  4. Click the celebration badge to open the animation editor
  5. Add up to 4 image layers with enter/exit animations
  6. Set frame timing for each layer's appearance
  7. Use the Test button to preview
  8. Toggle celebrations on/off with the circle button (top-right)
  9. Quick-play with the play button (top-left)

Image Requirements

  • Size: 1024 x 768 pixels (landscape)
  • Format: PNG with transparent background
  • Style: Dramatic, centered, no ground/shadows
  • Layers: Battlecry, Effect, Attack, Impact
